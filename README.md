Receitas Doces (Sistema de Receitas)

Projeto focado em um site de receitas de doces, desenvolvido para praticar conceitos de desenvolvimento web, banco de dados e controle de versão.

 Objetivo do Projeto
O sistema tem como foco principal listar várias receitas de doces saborosos e permitir que os usuários cadastrem novas receitas na plataforma.

 Tecnologias que pretendo usar:
* **Backend:** C# / ASP.NET Core
* **Frontend:** HTML, CSS e Bootstrap
* **Banco de Dados:** SQL Server
* **Controle de Versão:** Git e GitHub
